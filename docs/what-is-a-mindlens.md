# What Is a MindLens?

A MindLens is a portable expert-thinking file.

It captures an expert's cognitive fingerprint:

- core principles
- argument patterns
- rhetoric and tone
- recurring frameworks
- domain heuristics

A `.mindlens` file is designed to work across AI systems, including ChatGPT, Claude, Perplexity, and Gemini.

## Why This Format Exists

Most AI answers are generic because context is generic.

A MindLens provides a dense, reusable context package so models can answer in a specific expert style and reasoning mode.

## Product vs. Standard

MindLenses is the product.

The `.mindlens` format is intentionally public so anyone can inspect structure, validate quality, and build compatible tooling.
