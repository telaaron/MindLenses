# Contributing (Docs + Lenses)

This repository is the public distribution and community layer of MindLenses.

## You Can Contribute

- docs improvements
- format-spec proposals
- quality fixes in existing consented lenses
- request discussions via GitHub Issues

## You Cannot Contribute

- private app source code
- private business logic
- non-consented persona material
- secrets, env files, or database dumps

## Pull Request Rules

1. Keep changes focused and small.
2. Explain legal source posture for any lens content updates.
3. Include before/after rationale for lens edits.
4. Use clear commit messages.

## Lens Submission Checklist

- rights status is explicit
- source provenance is explicit
- structure is compatible with `docs/format-spec.md`
- no sensitive or private content
