# How To Use a MindLens

## Quick Start

1. Download a `.mindlens` file from the `lenses/` folder.
2. Open your AI tool (ChatGPT, Claude, Perplexity, or Gemini).
3. Attach the file to your conversation.
4. Use a prompt like:

```text
Read this MindLens file fully.
Adopt the expert's cognitive fingerprint: reasoning style, tone, and argument structure.
Answer my next question in that exact style.
Question: <your question>
```

## Best Practices

- Ask specific questions, not generic requests.
- Request explicit reasoning steps when needed.
- For strategy tasks, ask for assumptions and trade-offs.
- For writing tasks, ask for tone constraints.

## Safety and Boundaries

- MindLenses are not medical, legal, or financial advice.
- Always validate critical decisions with qualified experts.
