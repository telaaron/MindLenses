# Contributing to MindLenses Public Repo

This repository is intentionally public and focuses on:

- Lens distribution (`lenses/`)
- open documentation (`docs/`)
- community requests (GitHub Issues)

For full contribution guidance, see `docs/contributing.md`.

## Issue First

For new lens ideas, open an issue using the request template.

## PR Scope

Accepted PR scope:

- docs and format clarifications
- improvements to existing consented lens files

Rejected PR scope:

- application backend/frontend source code
- sensitive business logic
- private data and unapproved source material
